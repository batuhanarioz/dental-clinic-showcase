# Dental Clinic CRM

> Full-stack clinic management platform built for dental practices.

🔗 **Live:** [clinic.nextgency360.com](https://clinic.nextgency360.com)

---

## Overview

A complete practice management system developed from scratch for a dental clinic. Covers the full patient lifecycle — from appointment booking to treatment completion and billing.

## Features

- **Patient Management** — Profiles, medical history, treatment records
- **Appointment Scheduling** — Calendar view, availability management, reminders
- **Doctor Panel** — Per-doctor dashboard, daily schedule, patient notes
- **Treatment Tracking** — Tooth chart, diagnosis history, treatment plans
- **Billing & Payments** — Invoice generation, payment status, receipts
- **Reporting** — Patient statistics, revenue reports, clinic performance

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | Next.js 14, TypeScript, Tailwind CSS |
| Backend | Supabase (PostgreSQL + Auth + Storage) |
| Deployment | Vercel |
| Auth | Supabase Auth (role-based: admin, doctor, secretary) |
